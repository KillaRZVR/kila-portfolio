# KILA Telegram Bot

Локальный Telegram-бот принимает голосовые и текстовые задачи, транскрибирует голос через Whisper и создаёт GitHub Issue в `KillaRZVR/kila-portfolio`.

## Установка

1. В Telegram открой `@BotFather`.
2. Отправь `/newbot`, укажи имя и username бота.
3. Скопируй выданный токен. Не отправляй его в чат и никому не показывай.
4. В папке `telegram-bot` запусти `setup-telegram-bot.bat`.
5. Вставь токен Telegram и GitHub token с доступом к Issues репозитория.
6. Установщик покажет код привязки.
7. В Telegram отправь новому боту `/start КОД`.
8. После ответа бота можно отправлять голосовые и обычный текст.

## Обычный запуск

Дважды нажми `start-telegram-bot.bat`. Окно должно оставаться открытым.

При первой голосовой задаче Whisper скачает модель `small` размером около 500 МБ. Токены и Telegram ID хранятся только локально и исключены из Git.
