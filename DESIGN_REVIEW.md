# KILLA — дизайн-ревью

## Основной сайт

- Production: https://kila-portfolio-six.vercel.app/
- Статическая версия без JavaScript: https://kila-portfolio-six.vercel.app/review.html

Если production-страница отображается белой в браузере агента, используйте `review.html`: это полный статический слепок ключевых экранов без React, JavaScript и анимаций.

## Что оценивать

1. **Первый экран** — позиционирование, читаемость оффера, заметность CTA и визуальная иерархия.
2. **Примеры работ** — убедительность кейсов и достаточность описаний для продажи услуги.
3. **Road map** — понятность процесса для потенциального заказчика и снижение тревожности перед заказом.
4. **Цены / Контакты** — ясность следующего шага и доверие к способам связи.
5. **Общее впечатление** — соответствие образу независимого дизайнера и разработчика, коммерческая убедительность, сильные и слабые места.
